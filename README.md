# 04.6 - local and session storage

## Part 1

1. Add type="date" to the line of HTML that has id="date"

2. Add the line of Javascript code to `script.js` that will prevent your page from refreshing when you hit the submit button (hint: has something to do with e/event._____())

3. Save the date value to a variable called date.

4. Save the date variable to the LOCAL STORAGE with the key named "local storage".

5. Save the riddle value to a variable called riddle.

6. Save the riddle value to the SESSION STORAGE with the key named "riddle".

7. In our form, choose a date and answer the riddle.

8. Log the answer to our riddle in the console FROM THE SESSION STORAGE

9. Take a screenshot of your local and session storage console data with your new entries and include the results in the project.
